Explicacion de cada tipo de ciberataque:

  • OAS: Significa "Ataque de Suplantación de Aplicaciones" en inglés (Application Spoofing Attack). Este tipo de ataque ocurre cuando un ciberdelincuente crea una aplicación falsa que se hace pasar por una aplicación legítima con el objetivo de engañar a los usuarios para que revelen información confidencial o realicen acciones indeseables.
    • WAV: Se refiere a "Ataque de Voz por Internet" en inglés (Voice over Internet Protocol Attack). En este caso, los atacantes aprovechan vulnerabilidades en los sistemas de telefonía por Internet (VoIP) para interceptar llamadas, realizar llamadas fraudulentas, manipular mensajes de voz o espiar conversaciones.
      • KAS: Es el acrónimo de "Ataque de Secuestro de Cuentas" en inglés (Account Takeover Attack). Este tipo de ataque ocurre cuando un atacante obtiene acceso no autorizado a una cuenta de usuario legítima, ya sea a través del robo de credenciales, técnicas de phishing u otras vulnerabilidades, con el objetivo de obtener información sensible o realizar acciones fraudulentas en nombre del usuario legítimo.
      • ODS: Significa "Ataque de Denegación de Servicio" en inglés (Denial of Service Attack). Este tipo de ataque tiene como objetivo sobrecargar un sistema o red con tráfico o solicitudes falsas, con el fin de dejarlo inaccesible para los usuarios legítimos, interrumpiendo así su funcionamiento normal.
        • IDS: Se refiere a "Ataque de Supresión de Intrusión" en inglés (Intrusion Detection System Attack). Los atacantes utilizan este tipo de ataque para evadir o eludir los sistemas de detección de intrusiones, que son sistemas de seguridad diseñados para identificar y prevenir accesos no autorizados a redes o sistemas.
          • BAD: Es el acrónimo de "Ataque de Día Cero" en inglés (Zero-Day Attack). Este tipo de ataque ocurre cuando los atacantes aprovechan una vulnerabilidad de software desconocida o no parchada para infiltrarse en sistemas o redes, antes de que el desarrollador del software tenga la oportunidad de corregir la vulnerabilidad.
          • MAV: Significa "Ataque de Malware Avanzado" en inglés (Advanced Malware Attack). Este tipo de ataque implica el uso de malware sofisticado y altamente evasivo para infiltrarse en sistemas o redes y obtener acceso no autorizado o robar información confidencial.
            • VUL: Se refiere a "Ataque de Explotación de Vulnerabilidades" en inglés (Vulnerability Exploitation Attack). Los atacantes utilizan este tipo de ataque para aprovechar las vulnerabilidades conocidas en el software o hardware de sistemas o redes con el objetivo de obtener acceso no autorizado o realizar acciones maliciosas.
              • RMW: Es el acrónimo de "Ataque de Manipulación de Red" en inglés (Network Manipulation Attack). Este tipo de ataque implica la manipulación de la infraestructura de red o de los protocolos de comunicación para interceptar, modificar o alterar el tráfico de red con el objetivo de obtener información sensible o realizar acciones indeseables
